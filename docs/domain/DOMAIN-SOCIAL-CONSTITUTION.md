# DOMAIN-SOCIAL-CONSTITUTION

## 소셜(Social) 도메인 핵심 원칙

### 1. 소셜 기능은 커핑 노트 중심
- 좋아요와 댓글은 커핑 노트(`cupping_notes`)에만 달린다.
- 원두나 프로필에 직접 달리지 않는다.
- 커핑 노트가 삭제되면 연결된 좋아요·댓글도 함께 삭제된다. 이는 의도된 동작이다.

### 2. 좋아요는 중복 불가
- 한 사용자는 하나의 커핑 노트에 한 번만 좋아요를 누를 수 있다 (UNIQUE 제약).
- 좋아요 취소는 해당 레코드를 DELETE한다.

### 3. 댓글은 내용 기반
- 댓글은 순수 텍스트로만 구성된다.
- 중첩 댓글(대댓글)은 현재 지원하지 않는다.
