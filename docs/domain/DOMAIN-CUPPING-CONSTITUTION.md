# DOMAIN-CUPPING-CONSTITUTION

## 커핑(Cupping) 도메인 핵심 원칙

### 1. 커핑 노트는 향미 경험 기록
- 커핑 노트(`cupping_notes`)는 사용자가 특정 원두를 시음한 경험을 기록한다.
- 커핑 노트 생성 시 원두 등록을 함께 진행한다.
- 동일한 원두라도 로스팅 일자(`roast_date`)가 다르면 다른 경험이므로, 로스팅 일자를 함께 기록한다.
- 향(aroma), 산미(acidity), 바디(body)를 0.5~5.0 점수(0.5 단위)로 평가하고, 자유 메모를 남길 수 있다.

### 2. 점수와 노트의 분리
- 원두에 대한 종합 평점(`bean_ratings`)과 커핑 노트는 별도 테이블로 관리한다.
- 종합 평점은 원두당 사용자별 1개만 허용된다 (UNIQUE 제약).
- 커핑 노트는 원두당 여러 개 작성 가능하다.

### 3. 공개 데이터
- 모든 커핑 노트와 평점은 공개로 조회 가능하다.
- 본인이 작성한 노트만 수정/삭제할 수 있다.
- 노트 삭제 시 연결된 좋아요·댓글도 함께 삭제된다. 핵심 데이터는 원두(beans)에 있으므로 허용된 동작이다.
